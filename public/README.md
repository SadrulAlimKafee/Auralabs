# Public Assets

This folder contains static assets like images, icons, and other files.

Add your images and static content here. Next.js will serve these files from `/public` in your application.

## Usage

Place files in this directory and reference them like:

```jsx
<img src="/images/example.jpg" alt="Example" />
```

## Recommended Structure

```
/public
  /images
    - hero-bg.jpg
    - case-study-1.jpg
    - team-member.jpg
  /icons
    - favicon.ico
    - social-share.jpg
  favicon.ico
  robots.txt
```
